# UniStay (Student Housing Booking System)

## Requirements
- .NET 8
- Angular 20
- SQL Server

## Installation
1. Clone the repository:
   For Back-End -> git clone https://github.com/kirollossami/Final-Project-DEPI.git
   For Front-End -> git clone https://github.com/bothinamarwan/SHBS-front.git

2. Install dependencies:
   ng install

3. Run the project:
   ng serve

Or access live at: https://unistay-shbs.vercel.app/
